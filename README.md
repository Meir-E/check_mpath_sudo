# check_mpath_sudo

# Install on linux
```
curl --ssl-no-revoke -L https://github.com/Meir-E/check_mpath_sudo/archive/refs/heads/main.zip > myfile.zip
unzip -j myfile.zip && chmod +x check_mpath_sudo.pl && rm myfile.zip
```
